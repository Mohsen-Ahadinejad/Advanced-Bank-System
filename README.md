# 🏦 سامانه متمرکز صندوق بانکی (Advanced Banking System)

یک نرم‌افزار جامع مدیریت صندوق و تراکنش‌های بانکی که با زبان **Python** و رابط کاربری **Tkinter** توسعه یافته است. این سیستم با تمرکز بر استانداردهای مهندسی نرم‌افزار، معماری ماژولار و کدهای توسعه‌پذیر (Scalable) طراحی شده است.

## 📐 معماری نرم‌افزار (Software Architecture)
این پروژه بر پایه معماری پیشرفته **Clean MVC (Model-View-Controller)** با رویکرد Vertical Slicing پیاده‌سازی شده است تا بالاترین سطح از Cohesion و پایین‌ترین سطح از Coupling را ارائه دهد.

* **لایه Model:** مدیریت موجودیت‌ها (`Customer`, `Account`, `Transaction`) و قرنطینه‌سازی کوئری‌های دیتابیس در الگوی Repository. 
* **لایه View:** رابط کاربری گرافیکی ماژولار که در آن هر تب گرافیکی (مانند داشبورد، انتقال وجه و افتتاح حساب) به یک فایل و کامپوننت مستقل شکسته شده است.
* **لایه Controller:** استفاده از الگوی Orchestration با محوریت `BaseController` برای هماهنگ‌سازی ساب‌کنترلرهای مستقل و مدیریت متمرکز خطاها و بروزرسانی‌های UI.

## ✨ ویژگی‌های کلیدی (Key Features)
* **رابط کاربری بومی‌سازی‌شده (RTL):** طراحی دقیق فیلدها و جداول (Treeview) با پشتیبانی کامل از چینش راست‌به‌چپ (Right-to-Left) و فونت استاندارد فارسی (Samim).
* **تراکنش‌های اتمیک (Atomic Transactions):** تضمین یکپارچگی داده‌ها (Data Integrity) در عملیات‌های حساسی مانند کارت‌به‌کارت با استفاده از سیستم Rollback در پایگاه داده **SQLite**.
* **امنیت داده‌ها (Security):** بهره‌گیری از الگوریتم **SHA-256** به همراه Salt اختصاصی برای هش کردن رمز عبور مدیر و پین‌کد مشتریان بانکی.
* **رفع وابستگی حلقوی (Circular Dependency Resolution):** استفاده هوشمندانه از قابلیت **Duck Typing** در پایتون برای ارتباط بین کلاس‌های `Account` و `Transaction` بدون نیاز به Importهای متقابل.
* **تجربه کاربری بهینه:** اعتبارسنجی آنی ورودی‌ها، سیستم نرمال‌سازی خودکار اعداد فارسی/عربی به انگلیسی و نمایش Tooltip برای راهنمایی کاربران.

## 🚀 راهنمای نصب و اجرا

۱. ابتدا مخزن را روی سیستم خود Clone کنید:
```bash
git clone [https://github.com/Mohse-Ahadinejad/Advanced-Bank-System.git](https://github.com/Mohse-Ahadinejad/Advanced-Bank-System.git)
```

۲. وارد پوشه پروژه شوید:
```bash
cd Advanced-Bank-System
```

۳. **(اختیاری اما پیشنهادی)** برای تولید داده‌های تستی (مشتریان، حساب‌ها و تاریخچه تراکنش‌ها) اسکریپت زیر را اجرا کنید:
```bash
python seed_db.py
```

۴. فایل اصلی برنامه را اجرا کنید:
```bash
python main.py
```

## 📂 ساختار پوشه‌ها

├── model/              # منطق تجاری، کلاس‌های پایه و لایه ارتباط با دیتابیس
├── view/               # رابط کاربری تفکیک‌شده بر اساس کامپوننت
├── utils/              # توابع کمکی امنیتی و نرمال‌سازی داده‌ها
├── widgets/            # ابزارهای گرافیکی سفارشی (مانند Tooltip)
├── seed_db.py          # اسکریپت تولید خودکار داده‌های تستی (Mock Data)
└── main.py             # نقطه ورود برنامه و محل تزریق وابستگی‌ها (DI)
